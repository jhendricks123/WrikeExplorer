# Get-WrikeExplorerConfig

## SYNOPSIS
Gets the module configuration including the AccessToken if set

## SYNTAX

```
Get-WrikeExplorerConfig [<CommonParameters>]
```

## DESCRIPTION
Gets the module configuration including the AccessToken if set.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-WrikeExplorerConfig
```

Gets the WrikeExplorer configuration data.
